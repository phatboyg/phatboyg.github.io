ABOUT 
**
DistinctPress is a simple, but incredibly stylish theme ideal for a personal blog, or even as a business or news theme. We have also used the massively powerful WP Customizer to allow you to control the look and feel of DistinctPress – changing the header, footer and button colour with ease. You can also apply your own custom header image in seconds.

** 
Features 
** 
Drop-down Menu, Theme Options, Custom Widgets, Sidebar and WP Customizer Support.

Installation 
**
1. Primary:
 = Login to your wp-admin area and go to Appearance -> Themes.
 = Select Install tab and click on Upload link.
 = Select theme .zip and click on Install now button.
 = If you have any errors, use alternate method.

2. Alternate:
 = Unzip the template file that you have downloaded.
 = Via FTP, upload the whole folder to your server and place it in the /wp-content/themes/ folder.
 = Do not change directory name.
 = The template files should be here now: /wp-content/themes/dist/index.php (for example).

3. Log into your WP admin panel and click on the Appearance tab.

4. Now click on DistinctPress theme to activate it.

5. Complete all of the required inputs on the isquar Options page (in the WP admin panel under Appearance Menu) and click "Save Changes".
